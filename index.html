♔ Chess Variants Display

A Clojure library for rendering responsive checkerboards as HTML SVG elements. The checkerboards can be customized with different dimensions and colors, with CSS-responsive styling.

Features: Generate any size checkerboard, control colors with CSS, fully responsive SVG output, pure Clojure implementation with no external dependencies.
